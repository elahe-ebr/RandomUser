# Koin-Rx-Mvvm-Sample
In this project, a list of users is displayed randomly, for which the following tools are used
* koin
* RxJava
* mvvm
* Repository Pattern
* Room
* Navigation Component
* Retrofit
* Fresco
* EventBbus
* Animation
